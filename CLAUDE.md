# Guyana Law MCP

## Quick Reference

- **Jurisdiction**: Guyana (GY)
- **Source**: mola.gov.gy
- **Language**: English
- **Tools**: 8 core (non-EU jurisdiction)
- **Template**: Dominican-law-mcp

## Development

```bash
npm run census     # Enumerate laws from mola.gov.gy
npm run ingest     # Download full text
npm run build:db   # Build SQLite DB
npm run build      # Compile TS
npm test           # Run tests
```
